# Catalog App
Catalog App is a mobile application built using Flutter and Vector X. It allows users to browse through a catalog of items, add them to a cart, and manage their shopping experience. The app also includes a login page for user authentication and the ability to remove items from the cart.

## Features
Browse through a catalog of items
Add items to the cart
Remove items from the cart
User authentication with a login page

## Installation
Clone the repository:

```git clone https://github.com/your-username/catalog-app.git```
### Navigate to the project directory:

```cd catalog-app```

### Install the dependencies:

```flutter pub get```
### Run the app:

```flutter run```

## Usage
Launch the app on your device or emulator.

You will be presented with a login page. Enter your credentials to log in or create a new account.

Once logged in, you can browse the catalog by scrolling through the available items.

To add an item to the cart, tap on the "Add to Cart" button displayed with each item.

To remove an item from the cart, navigate to the cart screen by tapping on the cart icon in the app's navigation bar. In the cart screen, tap on the "Remove" button next to the item you wish to remove.
